# 📌 Overview

Mental wellness is just as important as physical health.

**Myndly** is an AI-powered journaling platform that enables users to track their emotional well-being through intelligent journal analysis, personalized AI insights, and an empathetic chatbot.

Instead of simply storing journal entries, Myndly understands emotions, detects mood trends, and encourages users to build healthier journaling habits through streaks and achievements.

# 🧠 AI Features

## 📖 Smart Journal Analysis

Every journal entry is automatically analyzed by AI to determine:

- Sentiment Score
- Emotional State
- Mood Category

---

## 🤖 AI Mental Wellness Chatbot

The chatbot provides:

- Emotional support
- Positive conversations
- Productivity tips
- Stress management
- Motivation
- Mindfulness guidance

---

## 📊 Weekly AI Insights

Based on journal entries from the previous seven days, Myndly generates:

- Overall emotional trend
- Recurring topics
- Personalized wellness recommendations

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/avisha191/myndly.git
```

---

## Backend

```bash
cd server
npm install
npm start
```

---

## Frontend

```bash
cd client
npm install
npm start
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET

GROQ_API_KEY=YOUR_GROQ_API_KEY

FRONTEND_URL=http://localhost:3000
```
